# Conversor de Moedas

Este é um projeto simples em Java que eu fiz para treinar programação.  
O objetivo é converter valores entre algumas moedas usando uma API de taxas de câmbio.

Ainda estou aprendendo, então o código não é perfeito, mas funciona e já me ajudou a entender melhor sobre Java, APIs e JSON.

---

## O que o programa faz

- Busca as taxas de câmbio atuais em uma API online  
- Permite escolher conversão entre:
  - USD → BRL  
  - BRL → USD  
  - EUR → BRL  
  - BRL → EUR  
- Mostra o valor convertido no final  

---

## Tecnologias usadas

- **Java 21**  
- **API ExchangeRate**  
- **Biblioteca Gson** (para ler JSON)  
- **IntelliJ IDEA**  

---

## Como rodar

1. Baixe ou clone este repositório  
2. Abra no IntelliJ (ou outra IDE de sua preferência)  
3. Adicione o arquivo **gson-2.10.1.jar** como dependência  
4. Execute o arquivo **Main.java**
